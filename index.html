• git clone path-to-directory
  • git status - состояние текущего локального репозитория
  • git add path-to-file - добавить файл пок контроль
  • git commit -am "added new file index.html"-зафиксировать изменения и дать кммит
  • git push - отправить все изменения в реозиторий
  • git pull - взять всё новое из репозитория
